Hi, I'm Feather!

Skills: Ruby / Rails / JavaScript / React / Redux / SQL / HTML / CSS / jQuery / Flux

I’m a web developer who recently moved to Manhattan. I’ve designed and coded backend and frontend work on several single-page web applications, including a Tumblr clone and a JavaScript browser game. I’m looking for a company where I can express my enthusiasm for new technologies and help build quality applications with staying power.

I’ve been drawn to math and the sciences for my entire life — when I was in elementary school, I had a lot of fun making a website where I could post my favorite recipes, and I’ve always enjoyed puzzle games and logic problems. When I studied biology in college, I helped develop science education applications for middle and high schoolers. It was really interesting to me to help translate materials from paper to interactive computer applications, and I liked the job of designing a user experience and figuring out what something would feel like from another person’s point of view.

After deciding I wanted to be a web developer, I enrolled at App Academy, a coding bootcamp with a less than 3% acceptance rate. At a/A, I designed and coded several fullstack projects. Check them out below!
